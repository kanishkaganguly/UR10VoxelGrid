Detailed Description

This class is a Gazebo World Plugin that is an intermediary between the VoxelGridPlugin and the ROS controller. It receives robot commands from gazebo::VoxelGridPlugin and relays them for execution to the ROSRobotController. Note that this architecture might be deprecated soon, in order to facilitate more direct communication between gazebo::VoxelGridPlugin and ROSRobotController. This class was done originally when ROS was not used and direct control of the robot through Gazebo was being attempted.
