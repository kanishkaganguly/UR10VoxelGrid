Detailed Description

This class interfaces with ROS to receive control commands from the gazebo::RobotControllerPlugin and relay that to the robot's JointTrajectory controller via a ROS publisher. This publishing is done via a separate thread, to prevent Gazebo locking up.

Examples:
tests/ros_controller_test.cpp.
